Algumas Tags iniciais

Para fazer o Título será utilizado a TAG <h1>

Para fazer um Parágrafo Ulize a TAG <p>

Para fazer a quebra de linhas utilize a TAG <br>

Para adicionar símbolos como esses: ← → ↓ → € ¥ ¢ ∅ $
Será necessário utilizar &entities;

Para colocar emojis, como esses aqui: 🥊 ✅ 👨
Será necessário colocar &#x(código do emoji)
